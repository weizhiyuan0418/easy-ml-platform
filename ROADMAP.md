# Roadmap

Easy ML Platform is an early-stage learning project. The roadmap is intentionally small and focused on making the local tabular workflow easier to understand and try.

## Short-Term Goals

- Improve first-run guidance for new users.
- Keep installer, README, screenshots, and Release assets consistent.
- Make import, training, and prediction errors easier to understand.
- Add small documentation improvements based on user questions.
- Keep tests and verification scripts reliable on Python 3.12, 3.13, and 3.14.

## Good First Issues

- Improve README wording or screenshots.
- Add small example datasets.
- Add tests for import, validation, or prediction edge cases.
- Improve bilingual UI messages.
- Report installer or startup issues with clear reproduction steps.

## Not Planned for Early Versions

- Production deployment features.
- User accounts, permissions, or team collaboration.
- Large-scale training infrastructure.
- Background training queues.
- Deep learning, image, text, or time-series modeling.

## Version Direction

- `v0.2.x`: stability, documentation, installer, and small usability fixes.
- `v0.3.x`: optional usability features such as clearer model summaries or simple data visualization.
- Later versions depend on feedback and available time.
