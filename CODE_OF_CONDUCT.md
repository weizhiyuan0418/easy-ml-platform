# Code of Conduct

This project follows a simple rule: be respectful, practical, and constructive.

## Expected behavior

- Discuss technical disagreements with evidence and context.
- Keep feedback specific to the work.
- Help new contributors understand project conventions.
- Respect privacy and do not publish private data, credentials, or proprietary datasets.

## Unacceptable behavior

- Harassment, threats, personal attacks, or discriminatory language.
- Publishing another person's private information.
- Repeated off-topic disruption.
- Submitting malicious code or intentionally unsafe instructions.

Maintainers may remove content or restrict participation when behavior harms the project or its contributors.
